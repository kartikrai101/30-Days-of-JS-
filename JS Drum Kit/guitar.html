GUITAR ROOM

Press the keys on your keyboard to hit different strings of the guitar.

Here's a guide to which key resembles which string:

  • E for E string (the topmost string)
  • B for B string (2nd string)
  • G for G string (3rd string)
  • D for D string (4th string)
  • A for A string (5th string)
  • E for E string (last string)
